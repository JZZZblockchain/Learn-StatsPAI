### `sp.rd2d()`

**2D boundary regression discontinuity estimation.**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `alpha` | `number` |  | `0.05` | Significance level for confidence intervals. |
| `approach` | `string` |  | `'distance'` | ``'distance'``: project onto signed distance to boundary, then apply univariate local polynomial RD. ``'location'``: fit bivariate local polynomial on each side of the boundary at evaluation points. |
| `boundary` | `string` |  |  | Function ``f(x1) -> x2`` defining the boundary curve. If None, the boundary is the vertical line ``x1 = 0``. |
| `bwselect` | `string` |  | `'mserd'` | Bandwidth selection method (used when ``h`` is None). |
| `data` | `string` | ✓ |  | Input dataset. |
| `eval_points` | `string` |  |  | Shape ``(k, 2)`` array of boundary evaluation points. If None, points are automatically selected along the boundary. |
| `h` | `number` |  |  | Manual bandwidth. If None, MSE-optimal bandwidth is selected. |
| `kernel` | `string` |  | `'triangular'` | Kernel function: 'triangular', 'uniform', or 'epanechnikov'. |
| `n_eval` | `integer` |  | `1` | Number of evaluation points when ``eval_points`` is None. Use 1 for a single pooled effect. |
| `p` | `integer` |  | `1` | Polynomial order for point estimation (1 = local linear). |
| `treatment` | `string` | ✓ |  | Binary treatment indicator (1 = treated, 0 = control). |
| `x1` | `string` | ✓ |  | First running variable (score dimension 1). |
| `x2` | `string` | ✓ |  | Second running variable (score dimension 2). |
| `y` | `string` | ✓ |  | Outcome variable name. |

**Example:**

```python
import statspai as sp
import pandas as pd

# Load or prepare data
```

### `sp.rowcount()`

**Row-wise count of non-missing values (Stata ``egen rownonmiss``).**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `data` | `string` | ✓ |  | pandas DataFrame containing the variables used by the estimator. |
| `vars` | `array[string]` | ✓ |  | vars parameter (Sequence[str]). |

**Example:**

```python
import statspai as sp
import pandas as pd

# Load or prepare data
df = sp.datasets.card_1995()  # example dataset

result = sp.rowcount(data=df, vars=["a", "b"])
print(result.summary())
```
