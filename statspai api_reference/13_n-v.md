### `sp.neural_causal_plot()`

**Plot diagnostics for TARNet/CFRNet/DragonNet results.**

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `alpha` | `number` |  | `0.72` | Significance level for confidence intervals and tests. |
| `ax` | `string` |  |  | ax parameter. |
| `bins` | `integer` |  | `30` | bins parameter (int). |
| `color` | `string` |  | `'#2563EB'` | color parameter (str). |
| `control_color` | `string` |  | `'#64748B'` | control_color parameter (str). |
| `figsize` | `string` |  | `[8, 5]` | figsize parameter. |
| `result` | `string` | ✓ |  | Neural causal result. |
| `title` | `string` |  |  | title parameter (Optional[str]). |
| `treated_color` | `string` |  | `'#DC2626'` | treated_color parameter (str). |
| `type` | `string (enum)` |  | `'cate'` | ``"cate"`` draws a CATE histogram, ``"effects"`` draws sorted unit-level CATEs, ``"propensity"`` draws DragonNet overlap, and ``"loss"`` draws per-epoch training/validation loss. |

> **type** options: `'cate'`, `'effects'`, `'propensity'`, `'loss'`

**Example:**

```python
import statspai as sp
import pandas as pd

# Load or prepare data
df = sp.datasets.card_1995()  # example dataset

result = sp.neural_causal_plot(result="value")
print(result.summary())
```
